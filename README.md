# README

* Список сущностей для проекта TestGuru:

```
1. Пользователь (субъекты с различными уровнями доступа)
2. Тест (объект для создания/прохождения пользователем)
3. Оценка (по факту прохождения теста присваивается прошедшему тест пользователю)
4. Вопрос (составляющая теста)
5. Вариант ответа (составляющая каждого вопроса теста)
6. Категория (делит тесты на пройденные/непройденные, тематическим категориям и тд.)
